# 智能匹配分析 - 模板与导出功能说明

## 功能概述

智能匹配分析页面现在包含完整的简历模板选择和多格式导出功能。

## 核心功能

### 1. 简历预览与模板选择
- **4种专业模板**：
  - 🎨 现代简约：适合互联网、科技行业
  - 💼 专业商务：适合金融、咨询行业  
  - ✨ 创意设计：适合设计、广告行业
  - 📄 极简主义：突出内容本身

- **实时切换**：在预览窗口中可以实时切换模板查看效果

### 2. 多格式导出
- **PDF** 📄：便携式文档，适合打印和分享
- **Word** 📝：Microsoft Word 文档，可编辑
- **HTML** 🌐：网页格式，可在浏览器查看
- **Markdown** 📋：纯文本格式，适合技术人员

### 3. 使用流程

1. **选择简历和职位** → 开始匹配分析
2. **查看匹配报告** → 总体匹配度 + 技能详情
3. **编辑优化建议**（可选）→ 人工调整 AI 建议
4. **勾选应用建议** → 选择要应用的建议
5. **生成优化简历** → 点击"生成优化后的简历"按钮
6. **预览与导出**：
   - 在"预览简历"标签页查看效果
   - 切换不同模板风格
   - 在"导出简历"标签页选择格式下载

## 技术实现

### 前端组件结构

```typescript
<Modal title="优化后的简历" width={1100}>
  <Tabs>
    {/* 预览标签页 */}
    <TabPane tab="预览简历" key="preview">
      <Select> {/* 模板选择器 */}
        <Option value="modern">现代简约</Option>
        <Option value="professional">专业商务</Option>
        <Option value="creative">创意设计</Option>
        <Option value="minimal">极简主义</Option>
      </Select>
      <div> {/* 简历内容预览 */} </div>
    </TabPane>
    
    {/* 导出标签页 */}
    <TabPane tab="导出简历" key="export">
      <Row>
        {/* PDF, Word, HTML, Markdown 导出卡片 */}
      </Row>
    </TabPane>
  </Tabs>
</Modal>
```

### 后端 API

#### 1. 生成简历
```
POST /api/v1/resume-generator/generate
{
  "resume_id": "简历ID",
  "job_id": "职位ID",
  "template": "modern",  // 模板选择
  "suggestions": [...]    // 应用的建议列表
}
```

#### 2. 导出简历
```
POST /api/v1/resume-generator/export
{
  "resume_data": {...},
  "format": "pdf",  // pdf, docx, html, markdown
  "filename": "优化简历.pdf"
}
```

#### 3. 下载文件
```
GET /api/v1/resume-generator/download/{filename}
```

## 注意事项

- 导出功能需要后端支持相应的文件生成库（如 xhtml2pdf, python-docx）
- PDF 导出需要处理中文字体问题
- 大文件导出可能需要较长时间，建议添加进度提示
- 导出的文件应该在服务器上设置自动清理机制

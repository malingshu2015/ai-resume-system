# 简历预览竖版布局更新

## 更新内容

### 1. 预览 Modal 布局调整

**之前**：横向宽屏布局（width: 90%）
**现在**：竖版 A4 纸张布局（width: 900px, centered）

### 2. 核心改进

#### 视觉效果
- ✅ 模拟真实 A4 纸张尺寸（210mm × 297mm）
- ✅ 白色背景 + 灰色外围，突出简历内容
- ✅ 添加阴影效果，增强立体感
- ✅ 居中显示，更符合阅读习惯

#### 技术实现
```typescript
<Modal
    width={900}
    centered
    bodyStyle={{ 
        padding: 0,
        maxHeight: '85vh',
        overflow: 'auto',
        background: '#f5f5f5'
    }}
>
    <div style={{ 
        display: 'flex', 
        justifyContent: 'center',
        padding: '24px'
    }}>
        <div style={{
            width: '210mm',      // A4 宽度
            minHeight: '297mm',  // A4 高度
            background: 'white',
            padding: '20mm',
            boxShadow: '0 2px 8px rgba(0,0,0,0.15)',
            borderRadius: '4px'
        }}>
            {/* 简历内容 */}
        </div>
    </div>
</Modal>
```

### 3. 响应式适配

#### 桌面端（>900px）
- 完整 A4 尺寸显示
- 居中对齐
- 最大高度 85vh，超出部分滚动

#### 移动端（≤900px）
- 自适应宽度（100%）
- 减小内边距（16px）
- 保持可读性

### 4. 打印优化

添加了打印样式，确保打印时：
- 移除阴影和圆角
- 移除内外边距
- 保持原始 A4 尺寸

```css
@media print {
    .resume-preview-page {
        box-shadow: none;
        border-radius: 0;
        padding: 0;
        margin: 0;
    }
}
```

### 5. 用户体验提升

**优点**：
1. **更真实**：预览效果接近最终打印/导出效果
2. **更专业**：竖版布局符合简历行业标准
3. **更清晰**：固定宽度避免内容过度拉伸
4. **更便捷**：可直接从预览界面打印

**使用场景**：
- 查看简历整体布局
- 检查内容排版
- 确认打印效果
- 快速预览导出结果

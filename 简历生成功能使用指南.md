# 智能简历生成与导出功能

## 📋 功能概述

智能简历生成功能可以基于AI优化建议，自动生成专业的简历文档，并支持多种模板和导出格式。

### 核心特性

- ✅ **AI 智能优化**：自动应用优化建议，提升简历质量
- ✅ **多模板支持**：4种专业模板（现代简约、专业商务、创意设计、极简主义）
- ✅ **多格式导出**：支持 PDF、Word、Markdown、HTML、JSON
- ✅ **实时预览**：生成前可预览效果
- ✅ **一键导出**：自动下载生成的简历文件

---

## 🎨 支持的模板

### 1. 现代简约 (Modern)
- **风格**：简洁现代的设计
- **适用行业**：互联网、科技、创业公司
- **配色**：蓝色系
- **特点**：清晰的层次结构，突出关键信息

### 2. 专业商务 (Professional)
- **风格**：传统专业的商务风格
- **适用行业**：金融、咨询、律师、会计
- **配色**：深蓝色系
- **特点**：正式严谨，适合传统行业

### 3. 创意设计 (Creative)
- **风格**：富有创意的设计
- **适用行业**：设计、广告、媒体、艺术
- **配色**：紫色系
- **特点**：视觉冲击力强，展现创造力

### 4. 极简主义 (Minimal)
- **风格**：极简风格
- **适用行业**：所有行业
- **配色**：灰色系
- **特点**：突出内容本身，适合内容丰富的简历

---

## 💾 支持的导出格式

### 1. PDF (推荐)
- **优点**：跨平台兼容，适合打印和分享
- **用途**：投递简历、打印
- **文件大小**：中等

### 2. Word (DOCX)
- **优点**：可编辑，方便后续修改
- **用途**：需要进一步编辑的场景
- **文件大小**：较小

### 3. Markdown
- **优点**：纯文本，版本控制友好
- **用途**：技术人员、GitHub 展示
- **文件大小**：最小

### 4. HTML
- **优点**：可在浏览器中查看，样式丰富
- **用途**：在线展示、个人网站
- **文件大小**：中等

### 5. JSON
- **优点**：结构化数据，便于程序处理
- **用途**：数据分析、系统集成
- **文件大小**：小

## 🚀 使用流程

### 步骤 1：分析简历
1. 上传或输入简历
2. 系统自动解析简历内容
3. AI 生成优化建议

### 步骤 2：生成简历
1. 选择模板（现代简约、专业商务等）
2. 点击"生成简历"按钮
3. 系统应用优化建议，生成新简历

### 步骤 3：预览效果
1. 点击"预览效果"按钮
2. 在弹窗中查看生成的简历
3. 确认效果满意

### 步骤 4：导出简历
1. 选择导出格式（PDF、Word等）
2. 点击"导出简历"按钮
3. 自动下载文件

## 📖 API 接口文档

### 1. 生成优化简历

```bash
POST /api/v1/resume-generator/generate
```

**请求体：**
```json
{
  "resume_id": "resume-id-123",
  "template": "modern",
  "optimization_suggestions": {
    "suggestions": [
      {
        "category": "内容优化",
        "item": "工作经历",
        "suggestion": "添加量化成果",
        "priority": "high"
      }
    ]
  }
}
```

**响应：**
```json
{
  "success": true,
  "message": "简历生成成功",
  "data": {
    "original_resume_id": "resume-id-123",
    "template": "modern",
    "template_info": {
      "name": "现代简约",
      "description": "...",
      "color_scheme": "blue"
    },
    "content": {
      "personal_info": {...},
      "work_experience": [...],
      "education": [...],
      "skills": {...}
    },
    "metadata": {
      "generated_at": "2026-02-07T08:00:00",
      "optimization_applied": true,
      "suggestions_count": 5
    }
  }
}
```

### 2. 导出简历

```bash
POST /api/v1/resume-generator/export
```

**请求体：**
```json
{
  "resume_data": {...},
  "format": "pdf",
  "filename": "张三_优化简历.pdf"
}
```

**响应：**
```json
{
  "success": true,
  "message": "简历导出成功",
  "file_path": "exports/张三_优化简历.pdf",
  "download_url": "/api/v1/resume-generator/download/张三_优化简历.pdf"
}
```

### 3. 预览简历

```bash
POST /api/v1/resume-generator/preview
```

**请求体：**
```json
{
  "resume_id": "resume-id-123",
  "template": "professional"
}
```

**响应：**
```json
{
  "success": true,
  "message": "预览生成成功",
  "data": {...},
  "html_preview": "<html>...</html>"
}
```

### 4. 获取模板列表

```bash
GET /api/v1/resume-generator/templates
```

**响应：**
```json
{
  "success": true,
  "templates": [
    {
      "id": "modern",
      "name": "现代简约",
      "description": "简洁现代的设计风格",
      "color_scheme": "blue"
    }
  ]
}
```

### 5. 获取导出格式

```bash
GET /api/v1/resume-generator/formats
```

**响应：**
```json
{
  "success": true,
  "formats": [
    {
      "id": "pdf",
      "name": "PDF",
```

## 📝 常见问题

### Q1: 生成的简历可以编辑吗？
A: 导出为 Word 格式可以编辑，PDF 格式不可编辑。

### Q2: 支持自定义模板吗？
A: 目前提供 4 种预设模板，后续可扩展自定义模板功能。

### Q3: 导出的文件保存在哪里？
A: 文件保存在 `backend/exports/` 目录，并提供下载链接。

### Q4: 如何添加新的模板？
A: 在 `resume_generator.py` 的 `TEMPLATES` 字典中添加新模板配置。

### Q5: 如何添加新的导出格式？
A: 实现对应的 `_export_xxx` 方法，并添加到 `EXPORT_FORMATS` 列表。

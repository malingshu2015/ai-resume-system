## 🚀 如何运行前端

### 1. 安装依赖（如果还没安装）
```bash
cd frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问应用
打开浏览器访问：http://localhost:5173

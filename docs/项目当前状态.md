## 📝 重要提示

1. **GLM API Key 是必需的**
   - 没有 API Key，AI 功能无法使用
   - 获取地址：https://open.bigmodel.cn/

2. **Docker 是推荐的运行方式**
   - 避免本地环境配置问题
   - 所有依赖都已在容器中配置好

3. **首次启动需要时间**
   - Docker 镜像构建约需 5-10 分钟
   - 请耐心等待

4. **开发模式支持热更新**
   - 前端代码修改自动刷新
   - 后端代码修改自动重载
